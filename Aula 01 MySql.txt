Aula 01 – MySQL 📌 (26/02/2025)
Origem dos Bancos de Dados
📌 Década de 1950: Antes da digitalização, os dados eram registrados manualmente em papel, organizados em pastas e armazenados em arquivos metálicos.

📌 Comparação com bancos de dados modernos:

Papel → Registros
Pasta → Tabelas
Armário → Arquivos
⚠️ Problema: O acúmulo excessivo de papel dificultava a organização e recuperação das informações.

A evolução da digitalização
✅ Anos 1950: Início da digitalização. Os dados passaram a ser armazenados sequencialmente em fitas magnéticas, com estrutura semelhante a um PDF, onde as informações eram registradas uma após a outra.

📌 Arquivos Sequenciais: Para acessar um dado específico, era necessário percorrer toda a sequência desde o início, tornando a recuperação lenta e ineficiente.

✅ Chegada dos discos rígidos (HDD):

Permitiu o armazenamento não sequencial dos dados.
Tornou possível acessar diretamente a informação desejada sem precisar percorrer toda a base.
Surgiram os índices, otimizando a busca de informações.
Esse modelo foi chamado de Arquivos de Acesso Direto.
A era dos bancos de dados
✅ Década de 1960: O Departamento de Defesa dos EUA criou o CODASYL, um grupo responsável pelo desenvolvimento de padrões para linguagens de programação e bancos de dados.

📌 Nesse encontro, surgiu a linguagem COBOL e foram discutidos conceitos fundamentais para o desenvolvimento dos bancos de dados modernos.

🛠️ Componentes de um banco de dados:

Base de Dados – Onde os dados são armazenados.
Sistema Gerenciador de Banco de Dados (SGBD) – Responsável pelo gerenciamento e segurança dos dados.
Linguagem de Exploração – Permite consultar e manipular os dados (ex.: SQL).
Programas Adicionais – Softwares auxiliares para otimização e administração.
📌 A IBM e a evolução dos bancos de dados
Na época, a IBM era referência em tecnologia (como o Google hoje) e apresentou os primeiros modelos de bancos de dados:

Modelo Hierárquico – Organização em estrutura de árvore.
Modelo em Rede – Permitia relações mais flexíveis entre os dados.
✅ Década de 1970: O matemático Edgar F. Codd propôs o modelo relacional, revolucionando a forma de armazenamento e acesso aos dados. Esse modelo é a base do SQL e dos bancos de dados modernos, como MySQL, PostgreSQL e SQL Server.
